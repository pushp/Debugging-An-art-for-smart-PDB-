Mode and Methods

  • Script mode
  • Run mode
  • Trace mode

Script Mode

    use -m pdb command
    execute python -m pdb example1.py

Run Mode

  • .run('statement')

    • pdb.run("myobj.myfunc")

  • .runeval(expression)

    • Same as run except it returns the value of evaluated code.

  • .runcall(function)

Trace Mode

    import pdb
    pdb.set_trace()
